## Service Management System (Hibernate Project)

### Project Description

The Service Management System is a Hibernate-based Java application developed to manage customer service operations efficiently.
This project demonstrates the implementation of Hibernate ORM (Object Relational Mapping) for performing database operations such as inserting, updating, deleting, and retrieving records.
Hibernate is used to simplify database interaction by mapping Java classes to database tables.

### Objectives
     - Implement Hibernate ORM framework
     - Perform CRUD operations using Hibernate
     - Manage customer service requests
     - Maintain technician and service ticket records
     - Demonstrate database persistence using Maven project structure

### Project Features
      - Customer Registration
      - Technician Registration
      - Service Ticket Creation
      - Database Persistence using Hibernate
      - Entity Mapping using Annotations
      - Session and Transaction Management
